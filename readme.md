

基于SSM的车辆租赁管理系统5拥有两种角色：管理员和用户

管理员：汽车类型管理、汽车管理、租赁管理、用户管理、续租管理、归还管理、保险管理、违章记录管理等

用户：登录注册、租赁汽车、归还和续租汽车、查看违章记录、收藏等
